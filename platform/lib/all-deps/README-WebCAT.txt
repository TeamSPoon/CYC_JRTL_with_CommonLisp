CycAppToolkitWeb
================

The Cyc Application Toolkit ("CAT") provides a series of modules to simplify the
process of building Cyc-enabled applications.

This module (the "WebCAT") provides for developing in a web/servlet environment.
It builds upon the CommonCAT.

For more information about the CAT, see:
  http://wiki.cyc.com/index.php/CycApplicationToolkit
